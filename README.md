# Glass Marbles

[![Glass Marbles Cover](./cover.jpg)](https://chiuhans111.github.io/marbles/)

An interactive, 3D glass marbles simulation. 

## Tech Stack

- **WebGL**: Real-time raytracing of 3D glass.
- **SVG Filters**: Composition.
- **Physics**: Rigid body dynamics.
